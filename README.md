### 第一堂课笔记

1. 生物信息学的定义
生物信息学是利用计算机、编程等多种手段研究生命信息的**编码、交互与调控**的学科。它是计算机科学、生物学与哲学学科交叉的结果。

2. 概念辨析：模型与算法

在课堂展示的 *"3. Modeling"* 环节中，这两个概念是计算生物学的基石，二者区别如下：

| 维度 | 模型 (Model) | 算法 (Algorithm) |
| :--- | :--- | :--- |
| **定义** | **是对现实问题的数学抽象或概率描述。**<br>它定义了变量之间的关系结构。 | **是求解模型参数或产生结果的具体计算步骤。**<br>它是一系列指令的集合。 |
| **关注点** | **"What" (是什么)**<br>我们如何用数学语言描述这个生物过程？ | **"How" (怎么做)**<br>我们如何通过计算找到最优解？ |
| **课件案例** | **线性回归方程** ($y = w_0 + \sum w_ix_i$)<br>**HMM (隐马尔可夫模型)**：定义了状态（Exon/Intron）和转移概率。<br>**神经网络架构**：定义了层数和连接方式。 | **排序 (Sorting)**：将无序数字变为有序。<br>**动态规划 (Dynamic Programming)**：用于序列比对或寻找 HMM 的最优路径。<br>**反向传播**：用于更新神经网络的权重。 |
| **比喻** | **地图**：描述了地形和路径网络。 | **导航逻辑**：在地图上计算出从 A 到 B 最短路径的方法。 |

3. 科学范式的演变
*   **第 4 范式 (Big Data-driven)**：从海量数据（如 TCGA, UK Biobank）中挖掘关联，不预设假设。
*   **第 5 范式 (AI-driven)**：利用 AI (如 AlphaFold, Transformer) 进行多模态、多尺度的建模与生成，加速科学发现。

### 学期学习计划

我准备遵循教学大纲中 **"Linux -> R -> Python/AI"** 的路线，重点培养**linux系统使用**的能力。

第一阶段：基础环境设置与Linux
*   **目标**：构建可复用的工作环境，摆脱图形界面依赖。
*   **关键行动**：
    *   [ ] 熟练掌握 Linux 命令行 (Bash)，理解文件权限与管道操作。
    *   [ ] 配置 **Docker** 环境，实现分析流程的容器化与可移植性。
    *   [ ] 建立 **GitHub** 仓库，养成用 Markdown 记录文档和备份代码的习惯。
    *   [ ] **考核点**：能够独立编写 Shell 脚本处理文本数据。

第二阶段：数据分析与统计绘图 (Weeks 5-12)
*   **目标**：掌握生物统计思维，能从杂乱数据中提取洞见。
*   **关键行动**：
    *   [ ] 精通 **R 语言** 基础，特别是 Tidyverse 数据清洗流程。
    *   [ ] 掌握 **NGS 数据分析** 流程（Mapping -> Quantification -> Visualization）。
    *   [ ] 复现经典论文中的图表（利用 `ggplot2`）。
    *   [ ] **重点**：理解 RNA-seq 中的差异表达分析原理，而非仅仅运行代码。

第三阶段：机器学习与项目实战 (Weeks 13-18)
*   **目标**：探索 AI 在组学中的应用，完成 Peer Learning 分享。
*   **关键行动**：
    *   [ ] 了解 **Transformer** 架构在序列分析中的应用（如类似 Chat-GPT 处理基因组序列）。
    *   [ ] **小组项目 (Peer Learning)**：
        *   选择一个具体课题（如 scRNA-seq 分析或深度学习模型应用）。
        *   运用费曼学习法，制作 PPT 并教会其他同学。
    *   [ ] 尝试阅读进阶书籍《Bioinformatics Data Skills》。
